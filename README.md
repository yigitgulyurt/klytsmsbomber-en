<h2>Installation</h2>
1.step
 
```console
git clone https://github.com/yiit06/klytsmsbomber-en.git
```
2.step

```console
cd klytsmsbomber-en
```
3.step

```console
pip3 install -r requirements.txt
```
4.step/computer

```console
python3 klyt-computer.py
```

4.step/phone

```console
python3 klyt-phone.py
```
